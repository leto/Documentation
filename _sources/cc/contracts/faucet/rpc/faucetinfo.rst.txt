**********
faucetinfo
**********

Displays the balance of the Faucet funds
========================================

Usage:
------

``faucetinfo``

Example Command:
----------------

.. code-block:: shell

    ./komodo-cli -ac_name=ATEST faucetinfo

Output:
-------

.. code-block:: json

    {
      "result": "success",
      "name": "Faucet",
      "funding": "200207.99860023"
    }
