*********
tokenlist
*********

This call will list all tokens that are available.

To find detailed info about any specific token, use ``tokeninfo`` command using the ``tokenid`` from the list.

Example Command:
----------------

.. code-block:: shell

    ./komodo-cli -ac_name=ATEST tokenlist
